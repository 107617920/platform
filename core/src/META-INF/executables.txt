The following executable files are shipped and installed with the standard LabKey Server Graphical Windows Installer.  They are also shipped in the zip download used for manual Windows installations.

1 Trans Proteomic Pipeline

Developer: {link:Institute for Systems Biology|http://www.systemsbiology.org}\\
License: {link:LGPL|http://www.gnu.org/copyleft/lesser.html}\\
Source: {link:Sashimi|http://sashimi.sourceforge.net/}

The Trans Proteomic Pipeline provides the following mass spectrometery analysis tools.

Raw mass spectrometer data conversion to mzXML
- ReAdW.exe - Requires ThermoFinnigan software

mzXML conversion to search engine input formats
- MzXML2Search.exe

Search engine native output conversion to pepXML
- Tandem2XML.exe
- MASCOT2XML.exe
- Out2XML.exe
- Sequest2XML.exe
    -- out2summary.exe
- Comet2XML.exe

Search validation tools
- PeptideProphetParser.exe
- ProteinProphet.exe

Quantitation tools
- ASAPRatioPeptideParser.exe
- ASAPRatioProteinRatioParser.exe
- ASAPRatioPvalueParser.exe
- LibraPeptideParser.exe
- LibraProteinRatioParser.exe
- Q3ProteinRatioParser.exe
- XPressPeptideParser.exe
- XPressProteinRatioParser.exe

Results batching and normalization tools
- xinteract.exe
    -- DatabaseParser.exe
    -- InteractParser.exe
    -- RefreshParser.exe
    -- batchcoverage.exe

1 STLPort

Developer: {link:STLPort Consulting|http://www.stlport.org/}\\
License: Public Domain\\
Source: {link:STLPort Project|http://sourceforge.net/projects/stlport}

The STLPort project provides a free open source Windows version of STL that is more portable to Unix versions of STL than Microsoft's version.  Used by ISB's Trans Proteomic Pipeline.

- stlport_vc846.dll

1 UnxUtils

License: Public Domain\\
Source: {link:STLPort|http://sourceforge.net/projects/unxutils}

The UnxUtils project provides a free open source port of most useful Unix commands.  The ones listed below are included with LabKey Server, and used by ISB's Trans Proteomic Pipeline.

- cat.exe
- find.exe
- ls.exe
- md5sum.exe
- mkdir.exe
- mv.exe
- rm.exe
- sort.exe
- tail.exe
- tar.exe

1 BsdTar

Developer: {link:GnuWin32|http://gnuwin32.sourceforg.net/}\\
License: {link:GnuWin32 License|http://gnuwin32.sourceforge.net/license.html}\\
Source: {link:GnuWin32 Project|http://sourceforge.net/projects/gnuwin32}

Bsdtar creates and manipulates streaming archive files (tar, tar.gz, tar.bz2, cpio, shar, iso, zip, gz and bz2 archives), and is used by ISB's Trans Proteomic Pipeline.

- bsdtar.exe
    -- bzip2.dll
    -- archive1.dll

License: {link:Artistic License|http://www.opensource.org/licenses/artistic-license.php}\\
Source: {link:X! Tandem|http://www.thegpm.org/TANDEM/index.html}

X! Tandem open source is software that can match tandem mass spectra with peptide sequences, in a process that has come to be known as protein identification.
